# Week Reviewer

This skill guides the agent to create a weekly review based on the daily notes from this week (sun-sat).

## Workflow

### 1. Review Notes

- **Review**: Review the notes white titles that fall under dates for this week
- **Consider a summary, take-aways**: From the reviewed notes, consider a summary and take-aways from the content

### 2. Create Weekly Review

- **Fill out weekly review template**: Use [the weekly review template](assets/weekly-review-template.md), and replace the example text with the information reviewed from the daily note for this current week (sun-sat)
- **Write file**: Write the finished template in the weekly-review/ directory in the current working directory

You are now finished with the review!
